# 🚀 LLM 课程学习计划

**课程**: [llm-course](https://github.com/mlabonne/llm-course) by Maxime Labonne  
**时长**: 4 周（20 天），每天约 1.5 小时  
**基础**: 985 理工科大三，学过高等数学/线代/概率论/Python

---

## 📅 总览

| 周 | 主题 | 天数 |
|------|------|------|
| 第1周 | LLM 架构：Transformer、注意力、分词、采样 | Day 1-5 |
| 第2周 | 数据集 + 监督微调 SFT | Day 6-10 |
| 第3周 | 偏好对齐 DPO/GRPO + 模型评估 | Day 11-15 |
| 第4周 | 量化 + 模型合并 + 综合实战 | Day 16-20 |

## 🔧 准备工作

1. 注册 Google 账号（用于 Colab）
2. 注册 Hugging Face 账号: https://huggingface.co
3. 打开 [Google Colab](https://colab.research.google.com) 确保能正常访问

## 📖 每天怎么用

1. 打开当天对应的 `day-XX.md` 文件
2. 按顺序完成 📖 阅读 → 💻 实操 → ✏️ 练习
3. 每完成一步就勾掉 ✅

## ⚠️ 重要

- 遇到不会的公式先跳过，不要卡住
- 优先跑代码，代码跑通了再回头补理论
- 每周末复习本周内容，不要欠债
